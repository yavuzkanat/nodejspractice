# Small Backend Projects with NodeJS

## [Project 1](backend-1/)
A simple website that allows users to send a message from the website and saves it to a file.

## [Project 2](backend-2/)
A shopping website built with the expressJS & Pug template engine using the MVC (Model - View - Controller) architecture.

## [Project 3](backend-3/)
A shopping website built with Express.js and the Pug template engine, following the MVC (Model–View–Controller) architecture. It uses Sequelize as the ORM to interact with an SQL database and provides a clean, modular structure for managing products, categories, and user interactions.
